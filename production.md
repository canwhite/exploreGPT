# microGPT 项目全局文档

## 项目定位

microGPT 是一个用纯 Python 实现的 GPT（Generative Pre-trained Transformer）模型，用于学习和理解 Transformer 架构的核心原理。该项目实现了一个完整的训练和推理流程，能够学习人名数据集并生成新的名字。

## 核心模块说明

### 1. Value 类（自动微分引擎）

**位置**: main.py:30-96

**功能**: 实现自动微分，支持反向传播

**核心方法**:

- `__add__`, `__mul__`, `__pow__`: 基本运算，构建计算图
- `backward()`: 反向传播，计算梯度

**数学原理**: 链式法则
$$\frac{\partial L}{\partial x} = \frac{\partial L}{\partial y} \cdot \frac{\partial y}{\partial x}$$

### 2. Tokenizer（分词器）

**位置**: main.py:14-27

**功能**: 将字符转换为 token ID

**数据流**:

- 输入: "anna"
- 输出: [BOS, 0, 13, 0, BOS]

**词汇表大小**: 27 (26个字母 + 1个BOS标记)

### 3. GPT 模型架构

**位置**: main.py:135-174

**核心组件**:

- Token Embedding (wte): 将 token ID 转换为向量
- Position Embedding (wpe): 编码位置信息
- Multi-Head Attention: 自注意力机制
- MLP: 前馈神经网络
- Residual Connection: 残差连接

**模型参数**:

```python
n_layer = 1          # 层数
n_embd = 16          # 嵌入维度
block_size = 16      # 上下文长度
n_head = 4           # 注意力头数
```

### 4. 训练循环

**位置**: main.py:183-216

**步骤**:

1. 前向传播: 计算预测值
2. 计算损失: 交叉熵损失
3. 反向传播: 计算梯度
4. 参数更新: Adam 优化器

**损失函数**:
$$L = -\log P(\text{target})$$

### 5. 推理生成

**位置**: main.py:221-234

**参数**:

- temperature: 控制生成随机性（0.5 = 适中）
- 采样策略: 按概率采样
