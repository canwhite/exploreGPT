# Linear 函数详解：矩阵乘法的完整指南

## 🎯 Linear 函数是什么？

**一句话**：Linear 函数就是**矩阵乘法**（也叫线性变换、线性投影）。

```python
def linear(x, weight):
    return x @ weight.T  # 矩阵乘法
```

---

## 💡 为什么要用 Linear 函数？

### 核心作用

把输入向量转换成一个新的表示。

**类比：翻译机**

```
中文：["猫", "在", "睡觉"]
   ↓ Linear 函数（翻译）
英文：["cat", "is", "sleeping"]

虽然还是3个词，但"表示"变了！
```

**神经网络中的类比**

```
输入：[0.5, -0.3, 0.8, ...]  # 16维向量
含义：当前是字母 'n'

   ↓ Linear 函数（转换）

输出：[0.23, -0.15, 0.67, ...]  # 16维向量
含义：这个字母适合做 Query/Key/Value
```

---

## 🔍 Linear 函数的代码

### 在 main.py 中的定义

```python
# main.py: 第 140 行左右
def linear(x, weight):
    """
    线性变换函数

    参数:
        x: 输入向量
        weight: 权重矩阵

    返回:
        输出向量
    """
    return x @ weight.T  # 矩阵乘法
```

**注意**：
- `.T` 表示转置（transpose）
- `weight.T` 就是把矩阵的行和列互换
- 这不影响理解，只是实现细节

## 📊 逐步拆解 Linear 函数

### 第1步：理解输入和输出

```python
# 输入向量 x（16维）
x = [x0, x1, x2, ..., x15]

# 权重矩阵 weight（16×16）
weight = [
    [w00, w01, ..., w015],  # 第0行
    [w10, w11, ..., w115],  # 第1行
    ...
    [w150, w151, ..., w1515]  # 第15行
]

# 输出向量（16维）
output = linear(x, weight)
       = [o0, o1, o2, ..., o15]
```

### 第2步：矩阵乘法如何计算

```python
# 计算第0个输出 o0
o0 = x0*w00 + x1*w10 + x2*w20 + ... + x15*w150
#   ↑      ↑      ↑             ↑
#   输入   第0列的所有元素

# 计算第1个输出 o1
o1 = x0*w01 + x1*w11 + x2*w21 + ... + x15*w151
#   ↑      ↑      ↑             ↑
#   输入   第1列的所有元素

# ... 依此类推

# 计算第15个输出 o15
o15 = x0*w015 + x1*w115 + x2*w215 + ... + x15*w1515
```

**关键观察**：
- 每个输出都是输入向量的**加权求和**
- 不同的输出使用不同的权重（weight的不同列）

## 🔢 完整数值例子

### 例子：简化版本（3维）

```python
# 输入向量 x（3维）
x = [2.0, 3.0, 4.0]

# 权重矩阵 weight（3×3）
weight = [
    [0.1, 0.2, 0.3],  # 第0行
    [0.4, 0.5, 0.6],  # 第1行
    [0.7, 0.8, 0.9]   # 第2行
]

# 计算 output = x @ weight.T
# (weight.T 就是把行和列互换)

# 第0个输出
o0 = 2.0*0.1 + 3.0*0.4 + 4.0*0.7
   = 0.2 + 1.2 + 2.8
   = 4.2

# 第1个输出
o1 = 2.0*0.2 + 3.0*0.5 + 4.0*0.8
   = 0.4 + 1.5 + 3.2
   = 5.1

# 第2个输出
o2 = 2.0*0.3 + 3.0*0.6 + 4.0*0.9
   = 0.6 + 1.8 + 3.6
   = 6.0

# 最终输出
output = [4.2, 5.1, 6.0]
```

### 验证维度

```
输入:  [2.0, 3.0, 4.0]        3维
权重:  3×3 矩阵
输出:  [4.2, 5.1, 6.0]        3维

维度保持不变（都是3维）！
```

## 🎨 为什么叫"线性"变换？

### 什么是"线性"？

**线性变换的两个性质**：

1. **可加性**：f(x + y) = f(x) + f(y)
2. **齐次性**：f(a·x) = a·f(x)

**用代码表示**：

```python
# 假设 f 是线性变换
def f(x):
    return linear(x, weight)

# 性质1：可加性
x1 = [1, 2, 3]
x2 = [4, 5, 6]

# 左边：f(x1 + x2)
left = linear([1+4, 2+5, 3+6], weight)
     = linear([5, 7, 9], weight)

# 右边：f(x1) + f(x2)
right = linear([1, 2, 3], weight) + linear([4, 5, 6], weight)

# 结果：left == right ✅

# 性质2：齐次性
a = 2.5

# 左边：f(a·x)
left = linear([2.5*1, 2.5*2, 2.5*3], weight)

# 右边：a·f(x)
right = 2.5 * linear([1, 2, 3], weight)

# 结果：left == right ✅
```

**非线性的例子**：

```python
# 这个不是线性变换！
def f(x):
    return x ** 2  # 平方运算

# 验证
f(x1 + x2) = (x1 + x2)^2 = x1^2 + 2*x1*x2 + x2^2
f(x1) + f(x2) = x1^2 + x2^2

# f(x1 + x2) ≠ f(x1) + f(x2) ❌
```

**记忆口诀**：

```
线性变换 = 只包含乘法和加法
         没有平方、开方、log等非线性运算
```

## 🔬 在 GPT 中的应用

### 应用1：计算 Query、Key、Value

```python
# 在注意力机制中
x = [0.5, -0.3, 0.8, ..., 0.2]  # 输入向量（16维）

# 三个不同的权重矩阵（都是16×16）
W_Q = matrix(16, 16)  # Query 权重
W_K = matrix(16, 16)  # Key 权重
W_V = matrix(16, 16)  # Value 权重

# 计算三个不同的表示
q = linear(x, W_Q)  # Query："我想查询什么？"
k = linear(x, W_K)  # Key："我提供什么信息？"
v = linear(x, W_V)  # Value："我的内容是什么？"
```

**为什么需要三个不同的矩阵？**

每个矩阵让输入关注不同的方面：

```
输入 X：一本书的内容

Q = linear(X, W_Q)  → Query："我要找什么？"
                  比如查询："关于深度学习的书"

K = linear(X, W_K)  → Key："这本书的关键词"
                  比如标签："AI, 神经网络, Python"

V = linear(X, W_V)  → Value："这本书的内容摘要"
                  比如内容："介绍 GPT 的原理"
```

### 应用2：MLP 中的线性变换

```python
# MLP（多层感知机）中
x = [x0, x1, ..., x15]  # 输入（16维）

# 第1层：放大（16→64）
W1 = matrix(64, 16)  # 权重矩阵（64×16）
h = linear(x, W1)    # 输出（64维）
h = relu(h)          # 激活函数（非线性）

# 第2层：缩小（64→16）
W2 = matrix(16, 64)  # 权重矩阵（16×64）
output = linear(h, W2)  # 输出（16维）
```

**维度变化**：

```
16维 → (linear + relu) → 64维 → (linear) → 16维
```

### 应用3：LM Head（最后的预测层）

```python
# 在 GPT 的最后
x = [x0, x1, ..., x15]  # 隐藏状态（16维）

# LM Head：16维 → 27维（27个token）
W_lm = matrix(27, 16)  # 权重矩阵（27×16）
logits = linear(x, W_lm)  # 输出（27维）

# logits = [logit0, logit1, ..., logit26]
# 每个数字表示对应token的分数
```

**维度变化**：

```
16维 → (linear) → 27维
```

## 🔤 .T（转置）是什么？

### 什么是转置？

把矩阵的行和列互换。

**例子**：

```python
# 原始矩阵
weight = [
    [1, 2, 3],
    [4, 5, 6]
]
# 2行3列

# 转置后
weight.T = [
    [1, 4],
    [2, 5],
    [3, 6]
]
# 3行2列
```

### 为什么要转置？

这是矩阵乘法的**惯例**：

```python
# 两种写法等价

# 写法1：不转置
output = x @ weight  # x 在前

# 写法2：转置
output = x @ weight.T  # weight 在前

# 只是参数存储的方式不同
```

**记忆口诀**：

```
.T 只是实现细节
不影响理解，知道是矩阵乘法就行
```

## 📊 Linear 函数 vs 传统写法

### 传统写法（不用矩阵）

```python
# 慢慢的写法
output = []
for i in range(16):
    sum = 0
    for j in range(16):
        sum += x[j] * weight[j][i]
    output.append(sum)
```

### Linear 函数写法（用矩阵）

```python
# 快速的写法
output = x @ weight.T  # 一行搞定
```

**对比**：

| 方面 | 传统写法 | Linear 函数 |
|------|---------|------------|
| **代码长度** | 多行循环 | 一行 |
| **执行速度** | 慢（CPU循环） | 快（GPU并行） |
| **可读性** | 难懂 | 简洁 |
| **性能** | 差 | 优 |

## 🎓 总结

| 符号 | 含义 | 作用 |
|------|------|------|
| **x** | 输入向量 | 原始数据 |
| **weight** | 权重矩阵 | 转换规则 |
| **output** | 输出向量 | 新的表示 |

**核心要点**：

1. **Linear 函数 = 矩阵乘法**
   ```
   output = x @ weight.T
   ```

2. **作用：转换表示**
   - 输入和输出维度可以相同
   - 但数值完全不同
   - 含义也不同

3. **线性变换的性质**
   - 可加性：f(x+y) = f(x) + f(y)
   - 齐次性：f(a·x) = a·f(x)

4. **广泛使用**
   - Query、Key、Value 计算
   - MLP 层
   - LM Head

**记忆公式**：

```
Linear 函数 = 矩阵乘法 = 线性变换

输入 @ 权重 = 输出

类比：
x = 原始照片
weight = 滤镜
output = 处理后的照片

虽然还是照片，但"风格"变了！
```

## 🔑 快速参考

### 常见问题

**Q: Linear 函数和非线性激活函数有什么区别？**

A:
- Linear：只做矩阵乘法，保持线性
- 激活函数（如ReLU）：引入非线性，让模型能学习复杂模式

**Q: 为什么有时候维度不变（16→16），有时候变（16→64）？**

A: 由权重矩阵的形状决定：
- 16→16：权重矩阵是 16×16
- 16→64：权重矩阵是 64×16
- 64→16：权重矩阵是 16×64

**Q: Linear 函数能改变输入的"含义"吗？**

A: 能！这正是它的作用：
- 输入：表示"这是一个字母"
- 输出：表示"这是否适合做 Query"

**Q: 为什么要用 .T（转置）？**

A: 这是实现惯例，不影响理解。只需要知道这是矩阵乘法就行。
